          检测到您曾经访问过本页面，浏览器可能缓存了旧版本的资源。

          建议您选择以下任意一种方式以确保使用最新版本：

          ① 按下 Ctrl + F5 强制刷新缓存
          ② 使用 无痕模式 / 隐私模式 打开本页面

          欢迎使用 TerraMap

          开始使用前，请点击上方 选择文件 按钮并打开您的泰拉瑞亚世界文件（.wld）。

          世界文件通常位于以下路径：

          • Windows: %USERPROFILE%\Documents\My Games\Terraria\Worlds
          • Windows (Steam 云存档): C:\Program Files (x86)\Steam\userdata\{用户ID}\105600\remote\worlds
          • MacOS: ~/Library/Application Support/Terraria/Worlds
          • MacOS (Steam 云存档): ~/Library/Application Support/Steam/userdata/{用户ID}/105600/remote/worlds
          • Linux: ~/.local/share/Terraria/Worlds
          • Linux (Steam 云存档): ~/.local/share/Steam/userdata/{用户ID}/105600/remote/worlds

          如遇BUG，可尝试 Ctrl+F5 清空网页缓存。

          使用前建议先前往bilibili查看2erTwo6的相关视频说明。

          您的浏览器不支持 HTML5 Canvas 标签。
