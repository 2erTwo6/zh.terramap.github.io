TerraMap Windows版

TerraMap Windows版是一个交互式的泰拉瑞亚 v1.4.4 世界地图查看器，加载迅速，允许您平移、缩放、查找方块、矿石、箱子中的物品、地牢、NPC等。

由于在下服务器资源有限，故不提供软件下载，请尝试前往Github进行下载，如遇无法访问的问题，可使用Watt Toolkit工具对Github加速。

功能特性

  • 完全只读，使用起来绝对安全，没有损坏或覆盖您世界文件的风险。
  • 通过预填充列表或手动浏览 .wld 文件，轻松打开您的泰拉瑞亚世界文件。
  • 交互式，使用鼠标和/或键盘快捷键进行平滑的平移和缩放
    • 按住鼠标左键拖动以平移
    • 使用鼠标滚轮缩放
    • 使用 - 和 + 键缩放
  • 使用“适应屏幕”和“缩放至100%”按钮。
  • 从下拉列表中选择一个方块或物品，或使用方块搜索窗口
    • 使用 Ctrl+F 键打开方块搜索窗口
  • 使用“查找下一个”和“查找上一个”按钮进行平移、缩放并选择匹配的方块，或包含所选物品的箱子
    • 使用 F3 键查找下一个
    • 使用 Shift+F3 键查找上一个
  • 使用“高亮全部”按钮高亮显示所有匹配的方块，或包含所选物品的箱子。
  • 右键单击箱子可查看其中物品的列表。
  • 右键单击告示牌或墓碑可查看其文字。
  • 使用“保存”按钮、文件->保存菜单项或 Ctrl+S 键保存您的完整世界地图的 PNG 图像。
  • 使用“刷新”按钮在游戏时重新加载并查看更改。
  • 使用“导航”菜单跳转到以下位置：
    • 出生点
    • 地牢
    • NPCs（商贩、机械师、军火商等）
  • 使用“世界属性”按钮或菜单查看信息，例如世界大小、月相、已击败的Boss、破坏的魔物之球、砸碎的祭坛等。
  • 支持命令行图像创建（非常适合服务器）。
